Сравнительный анализ численных методов решения уравнений Ляпунова
Этот проект содержит инструменты на языке MATLAB для сравнения эффективности и точности различных алгоритмов решения дискретных матричных уравнений Ляпунова.

Описание методов
В проекте исследуются три основных подхода:

Метод Малышевой: Итерационный алгоритм на основе QR-разложения.

Метод S-S(D-M): Комбинированный метод с использованием разложения Шура.

Метод S-S(K): Оптимизированный метод с прямым треугольным решением.

Основные файлы
S_S_K_and_M_and_S_S_D_M_tim.m — Алгоритм расчета времени и невязок для трех методов.

solve_discrete_triangular.m — Функция прямого решения для треугольных матриц.

tim_c_malshv.m — Главный скрипт для запуска тестов и построения графиков.

Инструкция по запуску
Для проведения эксперимента и автоматического построения графиков используйте функцию tim_c_malshv.

% Введите максимальную размерность матрицы (например, 100)
   tim_c_malshv(100);
